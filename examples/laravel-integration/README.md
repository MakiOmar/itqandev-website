# Laravel Integration Example

This directory contains example Laravel code to integrate with Qwik Dashboard.

## Structure

- `routes/` - Example API routes
- `controllers/` - Example controllers
- `resources/` - Example API resources
- `middleware/` - Example middleware configurations

## Setup

1. Copy the relevant files to your Laravel project
2. Adjust namespaces and paths as needed
3. Follow the main [LARAVEL_INTEGRATION.md](../../docs/LARAVEL_INTEGRATION.md) guide

## Files

- `routes-api.php` - Example API routes
- `AuthController.php` - Authentication controller
- `UserController.php` - User management controller
- `UserResource.php` - User API resource
- `cors-config.php` - CORS configuration example
